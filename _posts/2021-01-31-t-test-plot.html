The derivation for this plot comes from the following equation, which can be reordered to solve for sample size or effect size: $$N = \frac{(\Phi^{-1}(\beta)+\Phi^{-1}(1-\alpha/2))^2}{(\tau^2/\sigma^2)\times\gamma(1-\gamma)}$$ Each variables is defined below:
Variable Description
\(N\) Total sample size in experiment
\(\gamma\) Percent of sample size in the treatment group
\(\tau\) Absolute effect size
\(\sigma\) Standard deviation
\(\alpha\) * Statistical significance
\(\beta\) * Statistical power
\(\Phi\) * Cumulative density function of the standard normal distribution
* Default values are 5% significance and 80% power, which have critical values (\(\Phi^{-1}\)) at 1.96 and 0.84, respectively.

The source for this derivation is "The Econometrics of Randomized Experiments" (Athey and Imbens, 2016) Section 7.1, Page 34.
